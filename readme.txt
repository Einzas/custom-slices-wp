== Description ==
Plugin para mostrar noticias en un slider integrado con Gutenberg y Elementor.

== Installation ==
1. Descarga e instala el plugin desde el repositorio de WordPress o sube el archivo ZIP.
2. Actívalo desde el panel de plugins.
3. Configura y utiliza el bloque "News Slides Slider" en Gutenberg o el widget en Elementor.
